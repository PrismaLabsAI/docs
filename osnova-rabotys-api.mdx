**API-документация PrismaLabs**

**Базовый URL:**

```javascript
https://api.prismalabs.ru/v1
```

**Авторизация:**

Для работы с API необходимо передавать ваш уникальный токен в заголовках каждого запроса. Токен можно получить в личном кабинете после регистрации.

```javascript
Authorization: Bearer YOUR_TOKEN_HERE
```

**Формат запроса:**

Отправляйте POST-запросы на эндпоинт `/compile` с JSON-телом, содержащим ваш промпт и выбранную модель.

```json
{

   "question": "Сделай сайт для кофейни с меню на 3 позиции и формой заказа",

   "model": "claude-3.5-sonnet"

}
```

**Параметры:**

- `question` (обязательный) — ваш запрос на естественном языке

- `model` (обязательный) — идентификатор модели из списка поддерживаемых (полный список доступен в документации)

**Формат ответа:**

При успешной обработке сервер возвращает JSON со следующим содержимым:

```json
{

   "id": 1294482,

   "status": "completed",

   "response": "JSON-промпт, скомпилированный под выбранную модель",

   "price": "4593 TOKENS"

}
```

**Поля ответа:**

- `id` — уникальный идентификатор запроса в системе

- `status` — текстовый статус выполнения

- `response` — готовый промпт в формате JSON, оптимизированный под выбранную модель

- `price` — количество потраченных токенов

**Коды ответов и статусы:**

API использует стандартные HTTP-коды с дополнительными кодами в теле ответа для специфических ситуаций.

| Код ошибки | Значение                                                     |
| ---------- | ------------------------------------------------------------ |
| 200        | Запрос успешно обработан. В теле вы получите готовый промпт. |
| 400        | Не хватает токенов для обработки запроса. Пополните баланс.  |
| 500        | Внутренняя ошибка сервера. Напишите в поддержку.             |
| 600        | Токен не действителен. Выпустите новый.                      |
| 700        | Аккаунт заблокирован. Напишите в поддержку.                  |
| 800        | Слишком много запросов. Достигнут лимит PRM.                 |

**Пример запроса на Python:**

```python
import requests

url = "https://api.prismalabs.ru/v1/compile"

headers = {

    "Authorization": "Bearer YOUR_TOKEN",

    "Content-Type": "application/json"

}

data = {

    "question": "Сделай сайт для кофейни с меню на 3 позиции и формой заказа",

    "model": "claude-3.5-sonnet"

}

response = requests.post(url, json=data, headers=headers)

result = response.json()

print(result["response"])
```

**Пример запроса на JavaScript:**

```javascript
const response = await fetch('https://api.prismalabs.ru/v1/compile', {

  method: 'POST',

  headers: {

    'Authorization': 'Bearer YOUR_TOKEN',

    'Content-Type': 'application/json'

  },

  body: JSON.stringify({

    question: 'Сделай сайт для кофейни с меню на 3 позиции и формой заказа',

    model: 'claude-3.5-sonnet'

  })

});

const data = await response.json();

console.log(data.response);
```

**Важные замечания:**

- Токены списываются только при успешной компиляции (код 200)

- Цена в токенах зависит от сложности запроса и выбранной модели

- Лимиты на количество запросов зависят от тарифного плана

- Все запросы логируются для улучшения качества сервиса
